## notice-crawling


한양대 에리카 캠퍼스의 관련된 공지들을 앱을 통해 편리하게 볼 수 있도록 합니다.

공지가 올 때 알람이 울리도록 합니다.

아직 작업중입니다.


```
npm i

npm start

```
